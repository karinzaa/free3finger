Internet Freedom


ในสถานการปัจบันมีมือมืดกำลังจะพยายามที่จะปิดบังปิดกั้นการเข้าถึงข้อมูลทาง internet เราจะต้องป้องกันไม่ให้เราปิดบังปิดกั้นการได้ง่ายๆ ดังนี้


★ เพื่อความง่ายเราจะแบ่งระดับความอิสระไว้ 3 ระดับ



Lv.1 ใช้แอปติดต่อสื่อสารที่ถูกปิดกันได้ยาก



⬥Telegram

[ระดับความยาก: ง่าย]



เป็นแอปแชทที่มีฟีเจอร์หลายๆอย่างเหมือนๆกับแอปดังอื่นๆที่น่าจะดังที่สุดตอนนี้แอปนี้ใช้รูปแบบการส่งและเข้ารหัส ข้อมูลแบบฉบับของตัวเองในชื่อ MTProto ซึ่งมีการเข้ารหัสระหว่างเซิร์ฟเวอร์และเครื่องผู้ใช้ ก่อนที่จะมีการรับ-ส่งข้อมูลหากัน ด้วยกุญแจเข้ารหัสหลายระดับ โดยตัวข้อความจะได้รับการเข้ารหัสแบบ SHA-256 ก่อนเป็นขั้นแรก และมีการประกอบกับอีกหลายกุญแจทั้งแบบ 64, 128 และ 256 บิตในขั้นถัดๆ ไป สุดท้ายแล้วจะใช้การเข้ารหัสแบบ AES-256 ก่อนส่งข้อมูลออกไปและระบบ network ที่ใช้ใน App หลักๆจะเป็น P2P (Peer-to-Peer) ที่ยากต่อการปิดกั้นแม้แต่รัฐบาลจันหรือรัสเซียเคยทำสงครามกับ App นี้แบนไปถึง 19,000,000 IP Adress สรุปยอมแพ้

MTProto 2.0 ที่ใช้ใน Telegram client ~ server

※ E2EE (End-to-end encryption) ไม่ได้เปิดไว้เป็นค่าเริ่มต้นจะต้องเปิด Secret Chats ก่อนถึงจะใช้ได้



⬥Signal

[ระดับความยาก: ง่าย]



โดยรวมมีความคล้ายกับ Telegram แต่มี protocal ด้านความปลอดภัยหนาแน่นกว่าที่ชื่อ Signal Protocol ที่ใช้เทคนิค Double Ratchet algorithm และใช้ Curve25519, AES-256 และ HMAC-SHA256 ริเริมและพัฒนาโดย Edward Snowden

※ E2EE(End-to-end encryption) ถูกเปิดไว้เป็นค่าเริ่มต้น



⬥Bridgefy

[ระดับความยาก: ปานกลาง]



แอพพลิเคชั่นตัวนี้จะใช้ระบบ Bluetooth Mesh System ในการค้นหาคนที่เล่น Bridgefy เหมือนกันรอบตัวเราในรัศมี 100 เมตร และเชื่อมต่อคนเหล่านั้นเข้าด้วยกันในห้องแชตสาธารณะโดยไม่จำเป็นต้องใช้อินเตอร์เน็ตหรือสัญญาณมือถือแต่อย่างใดต่อให้เน็ตล่มหรือ โดนตัดสัญณานก็ไม่ได้รับผลกระทบเท่าไหร่แต่มี

※ ข้อจำกัดคือต้องมีคนท Bridgefy เหมือนกันรอบตัวเราในรัศมี 10~100 เมตรไม่งั้นจะใช้ไม่ได้
