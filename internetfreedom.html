Internet Freedom


ในสถานการณ์ปัจบันมีมือมืดกำลังจะพยายามที่จะปิดบังปิดกั้นการเข้าถึงข้อมูลทาง internet เราจะต้องป้องกันไม่ให้เราปิดบังปิดกั้นการได้ง่ายๆ ดังนี้



★เพื่อความง่ายเราจะแบ่งระดับความอิสระไว้ 3 ระดับ



Lv.1 ใช้แอปติดต่อสื่อสารที่ถูกปิดกันได้ยาก



⬥Telegram

ระดับความยาก:ง่าย



เป็นแอปแชทที่มีฟีเจอร์หลายๆอย่างเหมือนๆกับแอปดังอื่นๆที่น่าจะดังที่สุดตอนนี้แอปนี้ใช้รูปแบบการส่งและเข้ารหัส ข้อมูลแบบฉบับของตัวเองในชื่อ MTProto ซึ่งมีการเข้ารหัสระหว่างเซิร์ฟเวอร์และเครื่องผู้ใช้ ก่อนที่จะมีการรับ-ส่งข้อมูลหากัน ด้วยกุญแจเข้ารหัสหลายระดับ โดยตัวข้อความจะได้รับการเข้ารหัสแบบ SHA-256 ก่อนเป็นขั้นแรก และมีการประกอบกับอีกหลายกุญแจทั้งแบบ 64, 128 และ 256 บิตในขั้นถัดๆ ไป สุดท้ายแล้วจะใช้การเข้ารหัสแบบ AES-256 ก่อนส่งข้อมูลออกไปและระบบ network ที่ใช้ใน App หลักๆจะเป็นกึ่ง server less และมี P2P (Peer-to-Peer) โดยถ้าใช้กับ VPN และ proxy จะมีประสิทธิภาพขั้นสูงสุดและยากต่อการปิดกั้นโดยกรณีนี้รัฐบาลรัสเซียเคยทำสงครามกับ App นี้แบนไปถึง 19,000,000 IP Adress สรุปยอมแพ้

MTProto 2.0 ที่ใช้ใน Telegram client ~ server

※E2EE (End-to-end encryption) ไม่ได้เปิดไว้เป็นค่าเริ่มต้นจะต้องเปิด Secret Chats ก่อนถึงจะใช้ได้

※จะมีประสิทธิภาพขั้นสูงสุดเมื่อใช้กับ VPN และ proxy



⬥Signal

ระดับความยาก:ง่าย



โดยรวมมีความคล้ายกับ Telegram แต่มี protocal ด้านความปลอดภัยหนาแน่นกว่าที่ชื่อ Signal Protocol ที่ใช้เทคนิค Double Ratchet algorithm และใช้ Curve25519, AES-256 และ HMAC-SHA256 ริเริมและพัฒนาโดย Edward Snowden

※E2EE(End-to-end encryption) ถูกเปิดไว้เป็นค่าเริ่มต้น



⬥Bridgefy

ระดับความยาก:ปานกลาง



แอพพลิเคชั่นตัวนี้จะใช้ระบบ Bluetooth Mesh System ในการค้นหาคนที่เล่น Bridgefy เหมือนกันรอบตัวเราในรัศมี 100 เมตร และเชื่อมต่อคนเหล่านั้นเข้าด้วยกันในห้องแชตสาธารณะโดยไม่จำเป็นต้องใช้อินเตอร์เน็ตหรือสัญญาณมือถือแต่อย่างใดต่อให้เน็ตล่มหรือ โดนตัดสัญณานก็ไม่ได้รับผลกระทบเท่าไหร่แต่มี

※ข้อจำกัดคือต้องมีคนท Bridgefy เหมือนกันรอบตัวเราในรัศมี 10~100 เมตรไม่งั้นจะใช้ไม่ได้



Lv.2 ใช้ VPN แบบง่าย



⬥1.1.1.1

ระดับความยาก:ง่าย ※อาจมีค่าใช้จ่าย



1.1.1.1 with WARP เป็น Application ที่ช่วยเพิ่มความปลอดภัยในการใช้อินเตอร์เน็ตมากขึ้นเวลาคุณไปต่อใช้ WIFI ชาวบ้าน และช่วยเพิ่มความเร็วในการเข้าเว็บไซต์ WARP เป็นบริการ VPN ที่ไม่ต้องการการลงทะเบียนข้อมูลส่วนตัวใดๆ ยกเว้นต้องการความเร็วเพิ่มเติมคือ WARP+ โดยคิดราคา 59 บาทต่อเดือน

1.1.1.1 คือ DNS ของ WARP โดย Cloudflare

※WARP+ ราตา 59 บาทต่อเดือน สามารถการความเร็วเพิ่มขึ้น

※จ่ายแล้วใช้ได้หลายเครื่องทั้ง Android, iOS, Windows และ MacOs

※บริการนี้ไม่ได้มีการ Hide Ip เพราะฉนั้นบริการหรือเว็ปปลายทางจะสามารถเห็น Ip เราได้



Lv.3 ใช้ Tor ⬥ท่าไม้ตาย

Tor หรือชื่อเต็มว่า "The Onion Router" คือบริการที่สร้างเพื่อให้ผู้คนสามารถท่องอินเตอร์เน็ตได้โดยไม่เปิดเผยตัวตน เป็นระบบแบบกระจายศูนย์ ซึ่งผู้ใช้จะต้องทำการเชื่อมต่อกับเครือข่ายของเครื่องตัวกลางอื่นๆ ก่อน เมื่อเปรียบเทียบกับการเชื่อมต่อถึงเว็บไซต์โดยตรง เมื่อทำการเชื่อมต่อโดยทั่วไป โดยวิธีดังกล่าวนั้นจะช่วยซ่อนไอพีแอดเดรสของคุณจากเว็บไซต์ที่คุณเข้าชม เนื่องจากคุณใช้วิธีการเชื่อมต่อผ่านเซิร์ฟเวอร์ต่างๆ แบบสุ่ม ก่อนที่จะเชื่อมต่อถึงเว็บไซต์เป้าหมาย ทำให้เว็บไซต์ดังกล่าวไม่สามารถตามรอยคุณได้

หลักการทำงานของ Tor network



⬥Tor Browser

ระดับความยาก:ง่าย ※ความเร็วอาจช้ามาก



Tor Browser เป็น Application ที่ช่วยเพิ่มความปลอดภัยในการใช้อินเตอร์เน็ตท่องอินเตอร์เน็ตได้โดยไม่เปิดเผยตัวตนและไม่โดนบล็อก



⬥Orbot

ระดับความยาก:ง่าย ※ความเร็วอาจช้ามาก



Orbot เป็น Application ที่การทำงานคล้ายกับ VPN แต่จะทำงานในเครือข่ายของ Tor แทน
